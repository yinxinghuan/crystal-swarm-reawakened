# Requirements

## 1. Overview

《Crystal Swarm: Reawakened》是一段 45–75 秒的竖屏互动视觉体验：玩家用一根手指划开暗水，唤醒并引导一群棱晶，最终把它们聚成一次可截图的星云绽放；没有分数、失败或长期养成。

目标玩家是在 Aigram 内容流里偶然刷到作品的移动端用户。核心承诺是：第一次触碰后 100 ms 内，画面会明确地“知道你来了”。

## 2. Visual Design

画面为 390 × 844 竖屏的深色无边界空间，主视觉集中在中下部 72% 区域，避免遮挡顶部状态区与底部平台手势区。首屏只有一枚暗淡棱晶与稀疏尘埃，短提示“划过黑暗，唤醒晶体”。

每次移动触点留下 2.2 秒的折射轨迹；轨迹附近的晶体依次点亮、聚集并牵出细丝。累计唤醒 42 枚晶体后，场景进入 5.4 秒峰值：晶体成环旋转，中心出现彩色星云，随后冻结为可截图画面。

界面只保留标题、一次性提示、静音、重启和峰值时的“留住这一刻”文字。功能按钮为细线 SVG 图标，触控目标 44 × 44 px。

## 3. Game Mechanics

- 初始有 64 枚晶体，分布在半径 30–210 px 的不规则环带中；其中 5 枚在首屏以 8% 亮度呼吸。
- 每一帧更新 64 枚晶体的位置、速度、亮度与拖影。激活半径为触点周围 86 px；滑动速度超过 0.12 px/ms 时，激活半径扩大至 114 px。
- 被触及的晶体在 80 ms 内从 0 亮度升到 1 亮度，向最近触点施加 0.15 的引力，同时保留 0.92 的速度阻尼。
- 每次新激活晶体增加 1 点唤醒进度。进度达到 18、32、42 枚时，分别把环境亮度、轨迹折射和彩色微粒密度提升到 1、2、3 档。
- 进度达到 42 枚且距离第一次交互超过 5 秒时，触发 5.4 秒峰值：晶体沿半径 96–186 px 的螺旋环重新排布，中心星云脉冲 3 次，之后进入无限的静止余韵。
- 20 秒内没有连续移动时，保持已唤醒状态与慢速环境呼吸；不会失败或自动重置。
- 每次重启使用新随机种子，晶体初始位置、颜色和最终构图均不同。
- 默认启用完整粒子档；当设备像素比超过 2 或 `prefers-reduced-motion` 启用时，背景尘埃从 180 降至 80，拖影从 18 段降至 8 段。

## 4. Controls

- Pointer / touch：按下立即留下一个折射点并激活半径内晶体；拖动连续绘制轨迹，轨迹每 16 ms 至少采样一次。
- 鼠标：按住拖动等同于触控；移动端不依赖 hover。
- 键盘：`R` 重启；`M` 切换静音；`Space` 在无焦点输入元素时产生一次中心脉冲，方便桌面体验。
- 静音和重启为 44 × 44 px 的独立按钮，不干扰画布手势。

## 5. Win / Lose Conditions

没有失败条件、计分或传统胜利。唤醒 42 枚晶体即视为完成本次体验，并显示“留住这一刻 / Hold this moment”4 秒后淡出；重启按钮可随时生成下一次构图。

如果 Canvas 2D 不可用，显示静态的深色棱晶背景、标题和可用的重试按钮；不会卡在加载状态。

## 6. Sound Effects

- 第一次触碰：一枚 240–420 Hz 的正弦/三角混合音，80 ms，音量 0.025。
- 每枚晶体唤醒：以 520–1,040 Hz 为范围的短钟声，90 ms，最多同时播放 4 声，音量 0.012。
- 阶段升级：低频 110 Hz 与高频 760 Hz 的柔和上行扫频，320 ms，音量 0.035。
- 峰值：三次间隔 700 ms 的 180–1,200 Hz 空灵上行音，每次 700 ms，音量 0.055。
- 默认不播放持续背景音乐；用户先交互后才初始化音频。静音只关闭声音，不影响视觉反馈。
